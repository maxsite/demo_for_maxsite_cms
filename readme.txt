Демо данные для MaxSite CMS
---------------------------
После установки MaxSite CMS вы можете установить несколько готовых записей, чтобы быстро наполнить свой сайт для демонстрации или тестирования.

Вначале загрузите каталог «demo-images» в каталог сервера «uploads»:
	ваш сайт
		uploads/
			demo-images/
				1.jpg
				2.jpg
				3.jpg
				... и т.д.

Эти изображения используются в текстах записей.

В каталоге «autopost» находятся файлы, которые вы должны скопировать на свой компьютер. Это обычные текстовые файлы в формате AutoPost, которые вы можете редактировать любым текстовым редактором, например Notepad, Notepad++, Notepad2, PSPad и т.д.

Для создания записей на сайте, перейдите в админ-панели на страницу cоздания записи и нажмите ссылку «AutoPost» (вашсайт/admin/auto_post). Перетащите мышкой файл с компьютера в это окно — запись будет автоматически создана. Так же вы можете перетащить одновременно множество файлов: все они будут последовательно обработаны. Если какой-то файл будет продублирован, то MaxSite CMS проигнорирует создание записи.
